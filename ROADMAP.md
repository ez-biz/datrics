## Planned

### 1. Scheduled Queries & Auto-Refresh
Run saved questions on a schedule (hourly, daily, weekly). Cache results so dashboards load instantly. Email digest of results on schedule.

### 2. Alerts & Notifications
Set threshold-based alerts on questions (e.g., "notify me when revenue drops below $10k"). Support email and in-app notifications. Alert history log.

### 3. Sharing & Permissions
Share individual questions or dashboards with specific users or groups. Public/private visibility toggle. View-only vs edit access levels. Shareable public links.

### 4. Export & Download
Export query results as CSV, Excel, or PDF. Export charts as PNG/SVG images. Scheduled email reports with attached exports.

### 5. Dashboard Filters & Parameters
Add dynamic filters to questions (date range picker, dropdown selectors). Dashboard-level filters that apply across all cards simultaneously.

### 6. Question Versioning & History
Track changes to queries over time. View diff between versions. One-click revert to previous version. Audit log of who changed what.

### 7. AI-Assisted Query Building
Natural language to SQL (e.g., "show me top 10 customers by revenue"). Query optimization suggestions. Auto-explain complex SQL queries.

### 8. Embeddable Charts
Embed individual charts or full dashboards in external apps via iframe. API tokens for programmatic access. White-label styling options.

### 9. Additional Database Support
Expand beyond PostgreSQL, MySQL, and SQLite. Add support for: BigQuery, Snowflake, ClickHouse, DuckDB, Microsoft SQL Server, Amazon Redshift.

### 10. Dashboard Templates
Pre-built dashboard layouts for common use cases: SaaS metrics, e-commerce analytics, marketing funnel, financial overview. One-click setup from template.
